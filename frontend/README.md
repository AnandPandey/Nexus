# PythonSearch Frontend

A React frontend for the Python search engine.

## Setup

1. Ensure the backend is running on `http://localhost:5000`. To start the backend:
   ```
   python main.py serve
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

The frontend will be available at `http://localhost:5173` (default Vite port).

## Features

- Search bar for querying the search engine
- Display search results with title, URL, snippet, and score
- Responsive design
- Real-time search with loading indicator
