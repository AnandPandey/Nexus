## 🚀 Overview

**NEXUS** is a custom-built search engine that allows users to perform fast and relevant searches on structured datasets. It demonstrates core concepts of **information retrieval, ranking algorithms, and web development**.

Unlike traditional search tools, NEXUS gives you full control over:
- Data
- Ranking logic
- Search behavior

---

## ✨ Features

- 🔎 Custom-built search engine (no external APIs)
- ⚡ Fast and optimized search queries
- 📊 Intelligent ranking system
- 🎯 Keyword-based filtering
- 🌐 Clean and responsive UI (dark theme)
- 📁 CSV dataset support
- 📈 Optional data visualization using Chart.js
- 🔄 Dynamic search results

## 🔎 How It Works

1. Dataset is loaded from CSV  
2. Data is preprocessed and indexed  
3. User enters search query  
4. Backend processes query:
   - Tokenization  
   - Matching  
   - Scoring  
5. Results are ranked and returned  
6. UI displays results dynamically  

---

## 🧠 Search Algorithm

The ranking system is based on:

- 📌 Keyword frequency  
- 📌 Relevance score  
- 📌 Position of keywords  
- 📌 Matching accuracy  

### Basic Flow:
Query → Tokenize → Match → Score → Sort → Display
